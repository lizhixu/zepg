## 朝歌
1. 在 java 环境下打开提供的软件
2. 软件需要朝歌厂家提供授权码
3. 将名为 `product` 的文件拷贝进 u 盘插在机顶盒上

## 创维

``` shell
adb remount
adb logcat -v time > /U盘路径/XXX.LOG
```
